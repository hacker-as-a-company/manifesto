# Contributing

Thanks for helping grow Hacker as a Company.

## How to Contribute

- Open an issue to discuss ideas or problems.
- Fork the repo, create a branch, and submit a PR.
- Keep changes focused and documented.


## Style

- Keep language clear, respectful, and actionable.
- Prefer concise, reusable templates and checklists.
- Use markdown headings and lists for scannability.
