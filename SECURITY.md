# Security Policy

The manifesto and templates are public, non-executable content. If you believe you’ve found a security or privacy issue in this repository (e.g., exposed credentials in a contribution), please report it.

## Reporting
- Open a private report via GitHub Security Advisories, or
- Email: m.oumedlouz@gmail.com

Please do not include live secrets or client data in issues or pull requests.

## Scope
- Documentation and templates. No production systems are operated from this repo.

## Responsible Disclosure
We appreciate ethical, good-faith reports. We aim to acknowledge within 7 days.
