# Hacker as a Company

> Legal & free.

[Read the Manifesto](./HACKER_AS_A_COMPANY_MANIFESTO.md)

## Project Structure
- Manifesto: `HACKER_AS_A_COMPANY_MANIFESTO.md`
- Charter: `CHARTER.md`
- License: `LICENSE` (CC BY-SA 4.0)
- Docs site: `docs/` (GitHub Pages)
- Templates: `templates/` (contracts, checklists, profiles, pricing)
- Contributing: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `AGENTS.md`

## Quick Start
- Star or fork to “sign” support.
- Use templates in `templates/` for client-ready work.
- Optional: signed commits → `docs/GPG_SIGNING.md`.
- Hash the manifesto for timestamped proof → `./scripts/hash_manifesto.sh`.
- Enable Pages (docs folder) in repo settings.
